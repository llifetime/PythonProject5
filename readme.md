# Проект по работе с базой данных вакансий HH.ru

Этот проект собирает данные о вакансиях и компаниях с сайта hh.ru и сохраняет их в базу данных SQLite для последующего анализа.

## Функциональность

- Сбор данных о компаниях через API HH.ru
- Сбор данных о вакансиях компаний
- Сохранение данных в базу данных SQLite
- Анализ данных: средняя зарплата, поиск по ключевым словам и т.д.
